# 독립형 장애보고서 호출기 사용 설명서

## 1. 개요
이 프로그램은 데이터베이스에 저장된 장애 이력 보고서를 쉽고 빠르게 조회할 수 있는 독립 실행형 뷰어입니다.

## 2. 설치 및 실행 방법

### 2.1. 필수 요구 사항
- Node.js (최신 LTS 버전 권장)
- `incident_reports.db` (장애 이력 DB)

### 2.2. 설치
1. 명령 프롬프트(CMD)를 엽니다.
2. 폴더 이동:
   ```bash
   cd "d:\01 공통\공사중\independent_report_viewer"
   ```
3. 패키지 설치:
   ```bash
   npm install
   ```

### 2.3. 실행
1. 프로그램 실행:
   ```bash
   npm start
   ```
2. 브라우저 접속: `http://localhost:8003`

## 3. 사용 방법
1. **목록 조회**: 왼쪽 패널에 모든 장애 이력이 최신순으로 표시됩니다.
2. **상세 보기**: 목록에서 원하는 항목을 클릭하면 오른쪽 패널에 상세 정보(원인, 현상, 조치, 원본 텍스트)가 표시됩니다.

## 4. 문제 해결
- **DB 연결 오류**: `incident_reports.db` 파일이 상위 폴더(`..`)에 있는지 확인하세요.
- **npm 오류**: Node.js가 설치되어 있는지 확인하세요. (`node -v` 명령어로 확인 가능)
